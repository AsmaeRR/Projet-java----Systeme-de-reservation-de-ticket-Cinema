#  Cinema Ticket Reservation System (Java)

##  Description
This project is a Java console-based application designed to manage a complete cinema ticket reservation system.
The application provides an interactive menu that allows users to view available movies, display showtimes, make reservations, manage customers, cancel reservations, and view system statistics.
This project was developed in an academic context to practice core Java concepts and Object-Oriented Programming (OOP).

---

## Features
- Display available movies
- Display available showtimes
- Create customer accounts
- Reserve cinema tickets
- View a customer’s reservations
- Cancel a reservation
- View statistics:
  - Number of movies
  - Number of theaters
  - Number of showtimes
  - Number of customers
  - Active reservations
  - Total revenue

##  Project Structure
The project is composed of the following classes:

Main.java → Application entry point (console menu)
CinemaService.java → Business logic management
Customer.java → Customer representation
Movies.java → Movie representation
Showtime.java → Showtime management
Theater.java → Cinema theater management
Reservation.java → Reservation management

##  Installation and Execution

2. Open the project in a Java IDE
3. Compile the Java files
